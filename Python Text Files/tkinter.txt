tkinter commands

1. to use tkinter: import tkinter OR from tkinter import *
2. to create  a window: window = tkinter.Tk()
3. to rename the title of the window: window.title("New title of the window")
4. widget to place the buttons: Button(<root>, <other parms>)
5. to draw shapes in GUI: Canvas(<root>,<other parms>)
6. to create the checkbuttons (select more than one option): Checkbutton(<root>,<other parms>)
7. to create input fields in the GUI: Entry(<root>,<other parms>)
8. to make containers: Frame(<root>,<other parms>)
9. to create a single line widgets like images, text,etc: Label(<root>,<other parms>)
10. to create menus in the GUI: Menu(<root>,<other parms>)
11. to organize the widgets in the block: pack()
12. to organize the widgets in table-like structures: grid()
13. to place thw widget at a specific required position: place()


14. dialog to let the user choose a color: tkinter.colorchoose
15. base class for the dialogs defined in the other modules: tkinter.commondialog
16. common dialog to allow the user to specify a file to open or save: tkinter.filedialog
17. utilities to help work with fonts: tkinter.font
18. access to standard Tk dialog boxes: tkinter.messagebox
19. text widget with a vertical scroll bar built in: tkinter.scrolledtext
20. basic dialogs and conveniece functions: tkinter.simpledialog
21. drag and drop support for tkinter: tkinter.dnd
22. turtle graphics in a Tk window: turtle
23. To make a widget in Tk: classCommand newPathname options
24. At object creation time, using keyword arguments: fred = Button(self, fg="red", bg="blue")
25. After object creation, treating the option name like a dictionary index:
	fred["fg"] = "red"
	fred["bg"] = "blue"
26. Use the config() method to update multiple attrs subsequent to object creation: fred.config(fg="red", bg="blue")
27. form of the bind method is: def bind(self, sequence, func, add=''):
28. to register the file handler callback function func: Widget.tk.createfilehandler(file, mask, func)
29. callback command: callback(file, mask)
30. to unregister a file handler: Widget.tk.deletefilehandler(file)
19. constants used in the mask arguments: 
 	tkinter.READABLE
	tkinter.WRITABLE
	tkinter.EXCEPTION
